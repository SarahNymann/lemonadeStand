# Lemonade Stand App 🍋

Denne applikation er lavet som en del af en React-opgave.
Appen simulerer en simpel Lemonade Stand, hvor brugeren kan tjene og bruge penge.

Profitten håndteres via Redux Toolkit, hvilket gør det muligt at administrere state globalt i applikationen og opdatere brugerfladen dynamisk, når værdien ændrer sig.

## Funktionalitet

- Appen starter med **Total Profit: $0**
- Klik på **Sell Lemonade $5** for at øge profitten med $5
- Klik på **Buy Lemons $2** for at reducere profitten med $2
- Brugerfladen opdateres dynamisk, når profitten ændres via Redux store

## Teknologier

- React
- Redux Toolkit
- React Redux
- Vite
- JavaScript

## Sådan kører du projektet

1. Installer dependencies
   npm install
2. Start udviklingsserveren
   npm run dev
3. Åbn appen i browseren på den adresse der vises i terminalen (typisk http://localhost:5173).
