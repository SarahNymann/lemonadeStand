# Lemonade Stand App 🍋

Denne applikation er lavet som en del af en React-opgave med fokus på brug af `useState`.
Appen simulerer en simpel Lemonade Stand, hvor brugeren kan tjene og bruge penge.

## Funktionalitet
- Appen starter med **Total Profit: $0**
- Klik på **Sell Lemonade $5** for at øge profitten med $5
- Klik på **Buy Lemons $2** for at reducere profitten med $2
- Brugerfladen opdateres dynamisk, når profitten ændres

## Teknologier
- React
- Vite
- JavaScript
